# Support

Thanks for using Causalontology.

- **Questions and discussion:** open a [GitHub issue](https://github.com/ai-university-aiu/causalontology/issues) with the `question` label, or start a discussion if enabled.
- **Bugs:** open an issue using the bug report template — a conformance-style reproduction (an input and the wrong output) is the fastest path to a fix.
- **Security issues:** do not open a public issue; follow [SECURITY.md](SECURITY.md).
- **Contributing:** see [CONTRIBUTING.md](CONTRIBUTING.md). The 119 conformance vectors are the gate.
- **Where things are:** the normative spec is under [`spec/`](spec/); the bindings under [`bindings/`](bindings/); distribution status in [PUBLISHING.md](PUBLISHING.md).
